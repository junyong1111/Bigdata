- p-value 0.05보다 작으면 → 대립
    - 모든 회귀 계수가 0이 아니다. → 회귀 모형은 유의하다.
- p-value 0.05보다 크면 → 귀무
    - 모든 회귀 계수가 0

회귀 계수가 하나라도 0이 아니라면 회귀식이 그려질 수 있다.

💡 전체 회귀 모형 ⇒ F 검정
각각의 회귀 계수 ⇒ t검정

</aside>
